# **TileDesk – Tile Factory Desktop Application**

**TileDesk** is a desktop application developed using **Java Swing** to support operations of a tile factory dealing with marble, granite, and other tile products. The system provides a structured graphical interface to manage factory data including **products, employees, customers, sales, and reports**.  

⚠️ **Project Status:** This project is currently **under development**. Some features are still being implemented, and **database connectivity** will be added in upcoming updates.

---

## **Features**

**Product Module**
- Add, view, update, and delete tile products
- Manage marble, granite, and other tile types

**Sales Module**
- Record and track sales transactions
- View sold product details

**Employee Module**
- Add and manage employee records
- View employee information

**Customer Module**
- Maintain customer details
- Manage customer data

**Reports**
- Generate basic system reports
- View product and sales information

**Settings**
- Application configuration options

**Logout**
- Securely exit the application
